## Ejecutar localmente

### Con Live Server

1. Clona o descarga este repositorio.
2. Abre la carpeta en Visual Studio Code.
3. Instala la extensión **Live Server** de Ritwick Dey si aún no la tienes.
4. Haz clic derecho sobre `index.html`.
5. Selecciona **Open with Live Server**.

El proyecto se abrirá normalmente en `http://127.0.0.1:5500`.

### Sin dependencias

También puedes abrir `index.html` directamente en un navegador moderno. El formulario es una demostración de frontend y no envía datos a un servidor.
